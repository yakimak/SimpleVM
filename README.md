# Эмулятор загрузчика ОС

### Основные компоненты программы
- Стековый процессор
- Ram и HardDriver
- File System
- C-строки, LazySequence для БСД
### Последовательность работы
1. Включение компьютера (computer.powerOn();)
2. Инициализация BIOS, а затем загрузчика ОС (bios.initialize(); bios.loadBootloader();)
3. Создание файловой системы (computer.getFileSystem())
4. Проверка работоспособности
